How to Run the API Locally

  1. Install Node.js:
    Download and install Node.js from nodejs.org if you don't have it already.
  2. Open a terminal or command prompt.
  3. Navigate to the project folder:
    cd c:\Users\jorda\Documents\careerfoundry\big-beautiful-movie-api\movie_api
  4. Start the server:
    node server.js
  5. Open your browser and visit:
    • http://127.0.0.1:8080/ (Home)
    • http://127.0.0.1:8080/documentation (Documentation)

Stop the server anytime by pressing Ctrl + C in the terminal.
