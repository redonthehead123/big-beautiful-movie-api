Movie API Documentation

How to Run the API Locally

  1. Install Project:
    Download Big Beautiful Movie API from https://github.com/redonthehead123/big-beautiful-movie-api.
  2. Open index.html from within the files or...
  3. Open your browser and visit:
    • http://127.0.0.1:8080/ (Home)
    • http://127.0.0.1:8080/documentation (Documentation)
    • http://127.0.0.1:8080/movies (Movies)
