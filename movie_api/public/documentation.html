Movie API Documentation

How to Run the API Locally

  1. Install Project:
    Download Big Beautiful Movie API from https://github.com/redonthehead123/big-beautiful-movie-api.
  2. Open index.html from within the files or...
  3. Open your browser and visit:
    • http://127.0.0.1:8080/ (Home)
    • http://127.0.0.1:8080/documentation (Documentation)
    • http://127.0.0.1:8080/movies (Movies)

API Endpoints

Request URL Response
GET /movies Returns a list of all movies (JSON)
GET /movies/:title Returns data about a single movie by title
GET /genres/:name Returns data (description) about a genre by name
GET /directors/:name Returns data about a director by name
POST /users Allows new users to register
PUT /users/:username Allows users to update their user info (username)
POST /users/:username/movies/:movieID Adds a movie to a user's list of favorites
DELETE /users/:username/movies/:movieID Removes a movie from a user's list of favorites
DELETE /users/:username Deregisters an existing user

Endpoint Details

Get All Movies

  • Description: Returns a list of all movies.
  • URL: /movies
  • Method: GET
  • Query Parameters: None
  • Request Body: None
  • Response Example: 
    [
  {
    "title": "Inception",
    "description": "A thief who steals corporate secrets...",
    "genre": { "name": "Thriller", "description": "..." },
    "director": { "name": "Christopher Nolan", "bio": "...", "birth": 1970 },
    "imageURL": "http://...",
    "featured": true
  },
  ...
]

Get Movie by Title

  • Description: Returns data about a single movie by title.
  • URL: /movies/:title
  • Method: GET
  • Query Parameters: :title (movie title in URL)
  • Request Body: None
  • Response Example: 
    {
  "title": "Inception",
  "description": "A thief who steals corporate secrets...",
  "genre": { "name": "Thriller", "description": "..." },
  "director": { "name": "Christopher Nolan", "bio": "...", "birth": 1970 },
  "imageURL": "http://...",
  "featured": true
}

Get Genre by Name

  • Description: Returns data (description) about a genre by name.
  • URL: /genres/:name
  • Method: GET
  • Query Parameters: :name (genre name in URL)
  • Request Body: None
  • Response Example: 
    {
  "name": "Thriller",
  "description": "Thriller is a genre characterized by..."
}

Get Director by Name

  • Description: Returns data about a director by name.
  • URL: /directors/:name
  • Method: GET
  • Query Parameters: :name (director name in URL)
  • Request Body: None
  • Response Example: 
    {
  "name": "Christopher Nolan",
  "bio": "Christopher Edward Nolan is a British-American film director...",
  "birth": 1970,
  "death": null
}

Register New User

  • Description: Allows new users to register.
  • URL: /users
  • Method: POST
  • Query Parameters: None
  • Request Body Example (JSON): 
    {
  "username": "johndoe",
  "password": "password123",
  "email": "john@example.com",
  "birthday": "1990-01-01"
}
  • Response Example: 
    {
  "username": "johndoe",
  "email": "john@example.com",
  "birthday": "1990-01-01",
  "favoriteMovies": []
}

Update User Info

  • Description: Allows users to update their user info (username).
  • URL: /users/:username
  • Method: PUT
  • Query Parameters: :username (current username in URL)
  • Request Body Example (JSON): 
    {
  "username": "johnsmith",
  "password": "newpassword",
  "email": "johnsmith@example.com",
  "birthday": "1990-01-01"
}
  • Response Example: 
    {
  "username": "johnsmith",
  "email": "johnsmith@example.com",
  "birthday": "1990-01-01",
  "favoriteMovies": []
}

Add Movie to Favorites

  • Description: Adds a movie to a user's list of favorites.
  • URL: /users/:username/movies/:movieID
  • Method: POST
  • Query Parameters: :username (username in URL), :movieID (movie ID in URL)
  • Request Body: None
  • Response Example: 
    {
  "message": "Movie has been added to favorites."
}

Remove Movie from Favorites

  • Description: Removes a movie from a user's list of favorites.
  • URL: /users/:username/movies/:movieID
  • Method: DELETE
  • Query Parameters: :username (username in URL), :movieID (movie ID in URL)
  • Request Body: None
  • Response Example: 
    {
  "message": "Movie has been removed from favorites."
}

Deregister User

  • Description: Deregisters an existing user.
  • URL: /users/:username
  • Method: DELETE
  • Query Parameters: :username (username in URL)
  • Request Body: None
  • Response Example: 
    {
  "message": "User has been deregistered."
}
